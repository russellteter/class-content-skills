# Writing Rules — Class Content Pre-Delivery Checklist

Run through this checklist before delivering any Class blog post or long-form content.

## Structure Checks

- [ ] Key Takeaways section present near top (4-6 bullets)?
- [ ] Subheading every 150-250 words?
- [ ] Subheadings are benefit-focused, not just topic labels? ("How Engagement Scoring Changes Instructor Behavior" not "Engagement Scoring")
- [ ] CTA at end with action verb + specific benefit?
- [ ] Word count in target range for content type?
- [ ] Post follows one of the four approved templates?

## Voice & Tone Checks

- [ ] Contractions used throughout? ("don't," "can't," "it's," "we've")
- [ ] Active voice in 90%+ of sentences? (Name the actor: "Volvo's training team improved..." not "Engagement was improved...")
- [ ] No sentences an actual person wouldn't say out loud?
- [ ] Problem acknowledged before solution presented?
- [ ] Class introduced as enabler after best practices, not as the hero of paragraph one?

## Evidence Checks

- [ ] At least 2 third-party research citations from independent sources? (Not the same survey cited twice)
- [ ] Every major claim has a source (research, customer quote, or specific data)?
- [ ] Customer quotes include full name, title, and company?
- [ ] Customer quotes reference a specific challenge or measurable result? (Not generic praise)
- [ ] Statistics include source attribution?
- [ ] No unsourced percentages or unattributed claims?
- [ ] If only one research source was available, is that gap flagged in handoff notes?

## Terminology Checks

- [ ] "VILT" spelled out on first use, abbreviated after?
- [ ] No banned words (see class-brand-voice terminology reference)?
- [ ] Features described by outcome, not feature name?
- [ ] "Built on Zoom and Teams" (not "integrates with")?
- [ ] "Purpose-built for training" used when differentiating?
- [ ] No competitor names in marketing content (position against category)?

## Formatting Checks

- [ ] Paragraphs 2-4 sentences (80% of paragraphs)?
- [ ] No paragraphs longer than 5 sentences?
- [ ] Bulleted/numbered lists where appropriate (not overused)?
- [ ] Bold text used sparingly (5-8 instances max per post)?
- [ ] One exclamation mark max (and only if warranted)?
- [ ] No emojis in blog/website content?

## AI Detection Checks (Critical — Run First)

Run these checks against every paragraph BEFORE any other checks. These patterns flag content as AI-generated. Ordered by frequency of occurrence (most common first):

- [ ] **No three-item lists in prose?** This is the #1 AI tell. Count items in every comma-separated list. Three items ("engagement, consistency, and compliance") must become two ("engagement and compliance") or get restructured entirely. Bullet lists and product name lists (Docebo, Cornerstone, Canvas) are exempt. Prose is not.
- [ ] **No -ly adverbs?** Delete "increasingly," "significantly," "effectively," "dramatically," "seamlessly," "ultimately," "consistently," "essentially." Find a stronger verb or cut. Two -ly words max per 1,500-word post, and only if no alternative exists.
- [ ] **No false agency?** No inanimate thing performing a human action. "The platform transforms" → name the person. "Documentation assembles itself" → "Program managers pull compliance docs from one source." "This data tells a story" → "Training managers read this data and adjust." Check every sentence: is a human doing the verb?
- [ ] No binary contrasts? ("It's not X. It's Y." — state Y directly)
- [ ] No throat-clearing openers? ("In today's rapidly evolving..." — cut to the point)
- [ ] No dramatic fragmentation? ("Engagement. Connection. Results." — complete sentences)
- [ ] No lazy extremes? ("Every organization..." — name the specific type)
- [ ] No performative emphasis? ("Let that sink in." — delete)
- [ ] No narrator-from-a-distance voice? ("People tend to..." — put the reader in the room)
- [ ] No em dashes? (Use commas or periods)
- [ ] No sentences starting with "What," "When," "Where" as a crutch? (Lead with subject or verb)
- [ ] Sentence lengths varied? (No three consecutive sentences of similar length)

## Paragraph Rhythm Check

Read the draft aloud (mentally). Check:

- [ ] Mix of short (2-3 sentence) and medium (4-5 sentence) paragraphs?
- [ ] No three paragraphs in a row that start the same way?
- [ ] Opening sentence of each paragraph varies in structure?
- [ ] Transitions between sections feel natural, not mechanical?

## SEO Check

- [ ] Primary keyword in headline, first paragraph, one subheading, and meta description?
- [ ] 2-3 secondary keywords used naturally in body?
- [ ] Meta description under 160 characters?
- [ ] 2-3 internal links to related Class content?
- [ ] 1-2 external links to authoritative sources?

## Final Quality Gate

Before delivering, ask:

1. Would an L&D director share this with their team? (Is it genuinely useful?)
2. Would a competitor read this and respect it? (Is the argument sound?)
3. Could someone read this and get value without buying Class? (Is it educational first?)
4. Does any sentence sound like it was written by a marketing tool? (Rewrite it.)
5. Would Russell read this and say "this sounds like us"? (Does it pass the voice test?)
