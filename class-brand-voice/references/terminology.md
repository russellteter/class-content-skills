# Class Technologies Terminology Bible

## Product & Platform Terms

| Preferred | Avoid | Notes |
|-----------|-------|-------|
| Class | Class Technologies | Use "Class Technologies" only in press releases, legal, and first formal reference |
| Class for Zoom | Class Zoom integration | Product name, capitalize both words |
| Class for Teams | Class Teams integration | Product name, capitalize both words |
| Class for Web | — | The browser-based version |
| virtual classroom | online meeting room, webinar tool | Class IS a virtual classroom, not a meeting tool |
| purpose-built for training | built for education | "Purpose-built" is Class's key differentiator phrase |
| built on Zoom and Teams | integrates with, works alongside | "Built on" = inside, not next to. This is the mental model. |
| learning layer | add-on, plugin, extension | Class adds a learning layer on top of meeting platforms |
| sits inside Zoom and Teams | connects to, bolts onto | Russell's preferred framing. The product is INSIDE, not NEXT TO. |

## Industry & Domain Vocabulary

### Learning & Training Terms

| Term | Usage |
|------|-------|
| VILT (Virtual Instructor-Led Training) | The category Class operates in. Spell out on first use, abbreviate after. |
| Engagement | Measurable participation — not just "being there." Class means active interaction: polls, chat, breakout participation, attention metrics. |
| Learner / Participant | Preferred over "student" in corporate/gov contexts. "Student" is fine for higher ed and K-12. |
| Instructor / Facilitator / Trainer | Use interchangeably based on audience. "Instructor" is most common on the Class website. "Trainer" in corporate. "Facilitator" in consulting contexts. |
| Breakout rooms | Two words, lowercase. |
| Synchronous / Asynchronous | Use when the distinction matters. "Live" and "on-demand" are simpler alternatives for general audiences. |
| Microlearning | Short, focused learning segments. Use when discussing post-training reinforcement. |
| Blended learning | Mix of synchronous and asynchronous. Use when discussing comprehensive learning programs. |
| Cohort-based | Training delivered to groups moving through content together. Key for enterprise academies. |
| LMS (Learning Management System) | Spell out on first use. Common LMS partners: Cornerstone, Docebo, D2L/Brightspace, Canvas, Blackboard. |
| Kirkpatrick Model | Four levels of training evaluation (Reaction, Learning, Behavior, Results). Reference when discussing measurement. |
| Cognitive Load Theory | The science of how much information learners can process. Reference when discussing instructional design. |
| Ebbinghaus Forgetting Curve | The rate at which information is forgotten without reinforcement. Reference in post-training engagement content. |

### Business/ROI Terms

| Term | Usage |
|------|-------|
| ROI / Return on investment | Use when speaking to L&D leaders and executives about proving training value |
| Measurable outcomes | Preferred over "results" — implies data-backed proof |
| Scalable | Always follow with specifics: "scalable to 500 participants per session" not just "scalable" |
| Compliance | Required training (regulatory, safety, onboarding). Heavy in government and healthcare content. |
| Upskilling / Reskilling | Career development vocabulary. Use in corporate talent content. |
| Onboarding | New employee training. High-frequency corporate use case. |
| Retention | Both learner retention (knowledge) and employee retention (talent). Clarify which when both could apply. |

## Banned Words & Phrases

These words trigger AI-detection and undermine Class's credible, measured voice:

| Never Use | Use Instead |
|-----------|-------------|
| Revolutionary | Describe the specific change |
| Cutting-edge | Describe what it does differently |
| Game-changing | Name the specific impact |
| Next-level | Describe the improvement specifically |
| Best-in-class | Name the specific advantage |
| Synergy | Name what works together and why |
| Leverage (verb) | "Use," "apply," "build on" |
| Holistic | Name the specific components |
| Robust | "Solid," "thorough," "comprehensive" |
| Ecosystem | Name the actual pieces |
| Innovative | Show the innovation; don't label it |
| Empower | Use sparingly; prefer "give [person] the ability to" or "let" |
| Transform / Transformation | Use only when describing a specific, quantified before/after |
| Seamless (overuse) | Use once per piece max. Replace with "without switching tools" or "in one place" |
| Scalable (standalone) | Always follow with specifics |
| World-class | Cut entirely or name the specific quality |
| Thought leader | Cut entirely |
| Solution (generic) | Name the actual thing: "the platform," "the analytics dashboard," "breakout room management" |

## Competitor & Category Language

**Do:** Position against "traditional meeting tools" or "general-purpose video conferencing" as a category.

**Don't:** Name specific competitors (Zoom and Teams are partners, not competitors). Don't mention Adobe Connect, Webex Training, or others by name in marketing content.

**Exception:** Customer quotes may reference competitors naturally. That's authentic.

**Framing:** "Meeting tools weren't designed for training" positions Class as solving a category gap, not beating a competitor.
